Інформаційна система обліку СТО

Title (EN)

Car Service Management Information System

Короткий опис

Веб-орієнтована ERP система для автоматизації процесів станції технічного обслуговування: облік клієнтів, автомобілів, замовлень та фінансових операцій.

Ключові слова

СТО, ERP, веб-додаток, облік, база даних, Node.js

Актуальність теми

Сучасні СТО потребують автоматизації процесів для підвищення ефективності роботи. Використання інформаційних систем дозволяє зменшити кількість помилок, оптимізувати облік та покращити обслуговування клієнтів.

Мета дослідження

Розробка веб-інформаційної системи для автоматизації обліку діяльності СТО з використанням сучасних технологій веб-розробки.

Основні завдання

  • Аналіз предметної області СТО
  • Проєктування структури бази даних
  • Розробка REST API
  • Створення веб-інтерфейсу
  • Тестування системи

Методологія

У роботі використано технології Node.js, Express, SQLite. Для тестування застосовано Jest та Cucumber. Архітектура побудована за принципом клієнт-сервер.

Очікувані результати

Створення функціональної системи обліку СТО, яка дозволить ефективно керувати клієнтами, автомобілями та замовленнями, а також демонструє повний цикл роботи ERP-системи.
